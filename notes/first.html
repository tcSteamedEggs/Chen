          • 添加 v-pre

            • 模板语法可以在 Markdown 中使用，同样可以在代码块中生效

          • 默认情况（保持原样）

          1 + 2 + 3 = {{ 1 + 2 + 3 }}
          • 下述代码会被 Vue 编译
          • 也就是说我们在 Markdown 中可以使用 Vue
            • 模板语法
          1 + 2 + 3 = 6
          • JS 代码高亮，下面代码不会被正确编译
          1 + 2 + 3 = {{ 1 + 2 + 3 }}
          • 列表标记
          <ol>
  <li>小明</li>
  <li>小李</li>
  <li>小红</li>
</ol>
          • 上述代码将会在 Markdown 中渲染成下述
          1. 小明
          2. 小李
          3. 小红
